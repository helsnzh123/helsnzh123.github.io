Welcome to LPU Library

  • LPU offers a huge Library System with 11 Branches
  • The Central Library located in block 37 is the biggest
  • LPU academic community is well catered by all departmental libraries.
  • There are more than 43 lakh books and e-books on all subject fields.
  • Library system subscribes number of journals, magazines and newspapers.
  • Central Library of the university also manages an Institutional Repository for theses and dissertations.
  • A good number of theses/dissertations and research reports are available in libraries.
  • Besides books, library system also possesses rare book collection, reference sources, audio-visual material etc.

Library Rules

  • Entry without I.D. Card is not allowed
  • Exit without I.D. Card is not allowed
  • Bags and other items except Laptops and Notebooks are to be deposited at the gate of the library
  • If a student or a teacher wishes to take a book home, then he/she has to issue that particular book
  • Issued Books have to be returned within the alloted time period
  • Eatables are not allowed inside the library
  • Silence and decorum is to be mainatined at all times
  • If a student is found harming the property of LPU then he/she will be subjected to punishment or fine.
  • Mobile Phones have to be kept on Silent inside the library
